Запрос по всем 17 таблицам mart показал:

- Все остальные таблицы: 0 строк, 0 дублей (пустые)

Запрос для воспроизведения:
```sql
SELECT 'dim_date' AS таблица, COUNT(*) AS строк, COUNT(*) - COUNT(DISTINCT date_day) AS дублей FROM mart.dim_date
UNION ALL SELECT 'dim_crop', COUNT(*), COUNT(*) - COUNT(DISTINCT crop_sk) FROM mart.dim_crop
UNION ALL SELECT 'dim_equipment', COUNT(*), COUNT(*) - COUNT(DISTINCT eq_sk) FROM mart.dim_equipment
UNION ALL SELECT 'dim_field', COUNT(*), COUNT(*) - COUNT(DISTINCT field_sk) FROM mart.dim_field
UNION ALL SELECT 'dim_nomenclature', COUNT(*), COUNT(*) - COUNT(DISTINCT nom_sk) FROM mart.dim_nomenclature
UNION ALL SELECT 'dim_warehouse', COUNT(*), COUNT(*) - COUNT(DISTINCT wh_sk) FROM mart.dim_warehouse
UNION ALL SELECT 'dim_shift', COUNT(*), COUNT(*) - COUNT(DISTINCT shift_sk) FROM mart.dim_shift
UNION ALL SELECT 'fact_downtime', COUNT(*), COUNT(*) - COUNT(DISTINCT id) FROM mart.fact_downtime
UNION ALL SELECT 'fact_field_costs', COUNT(*), COUNT(*) - COUNT(DISTINCT id) FROM mart.fact_field_costs
UNION ALL SELECT 'fact_fuel_usage', COUNT(*), COUNT(*) - COUNT(DISTINCT id) FROM mart.fact_fuel_usage
UNION ALL SELECT 'fact_harvest', COUNT(*), COUNT(*) - COUNT(DISTINCT id) FROM mart.fact_harvest
UNION ALL SELECT 'fact_production_output', COUNT(*), COUNT(*) - COUNT(DISTINCT id) FROM mart.fact_production_output
UNION ALL SELECT 'fact_storage_balance', COUNT(*), COUNT(*) - COUNT(DISTINCT date_day) FROM mart.fact_storage_balance
UNION ALL SELECT 'fact_tmc_usage', COUNT(*), COUNT(*) - COUNT(DISTINCT id) FROM mart.fact_tmc_usage
UNION ALL SELECT 'fact_putevoy_rabota', COUNT(*), COUNT(*) - COUNT(DISTINCT doc_id) FROM mart.fact_putevoy_rabota
UNION ALL SELECT 'fact_spisanie_materialov', COUNT(*), COUNT(*) - COUNT(DISTINCT doc_id) FROM mart.fact_spisanie_materialov
UNION ALL SELECT 'fact_vypusk_urozhaya', COUNT(*), COUNT(*) - COUNT(DISTINCT doc_id) FROM mart.fact_vypusk_urozhaya
ORDER BY таблица;
```

```bash
docker exec -it ldali-airflow-web bash -c "find /opt/airflow/logs/dag_id=dag_extract_dvizhenie -name '*.log' -type f | sort | tail -1 | xargs tail -50" 2>/dev/null
```

Проверить логи dag_transform_mart:
```bash
docker exec -it ldali-airflow-web bash -c "find /opt/airflow/logs/dag_id=dag_transform_mart -name '*.log' -type f | sort | tail -1 | xargs tail -50" 2>/dev/null
```

Проверить строки в raw после запуска DAGов:
```sql
SELECT relname, n_live_tup AS rows
FROM pg_stat_user_tables
WHERE schemaname = 'raw'
ORDER BY rows DESC;
```
